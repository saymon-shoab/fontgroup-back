## Run Locally

Clone the project

```bash
  git clone https://github.com/saymon-shoab/fontgroup-back.git
```

Go to the project directory

```bash
  cd fontgroup-back
```

node js version

```bash
  nvm use 20.12.1
```
Install dependencies

```bash
  npm install
```
